## ğŸš€ Getting Started

### Prerequisites

- Node.js 22+ (see `.nvmrc`)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/asimov-systems/asimov.systems
   cd asimov.systems
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
